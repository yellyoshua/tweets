# Tweets Deploy - Twitter cron job bot

This is tweet bot that post a tweet every day.

## Technologies used

- [🤖] GitHub Actions (CI/CD)
- [💻] Javascript Nextjs (React)
- [📦] Vercel (Deployment)
- [🌐] Cloudflare Workers (Serverless)
- [✅] Cloudflare Domain (DNS)
- [🔒] Cloudflare SSL (SSL)
- [🔗] Twitter API (API)
- [🔃] MongoDB (Database)
